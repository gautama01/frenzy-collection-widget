Setup Instructions:

  1. Create a .env file in the project root with your Shopify credentials:
    2. VITE_SHOPIFY_DOMAIN=your-shop.myshopify.com
VITE_STOREFRONT_TOKEN=your-storefront-access-token
VITE_API_VERSION=2024-04
  2. Run npm run build to build the widget
  3. Copy the built files from dist/ to your web server
  4. Include the widget CSS and JS files in your page
